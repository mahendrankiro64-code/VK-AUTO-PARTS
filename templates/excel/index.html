Excel / Google Sheets Import & Export

Google Sheets tip: this app doesn't connect live to Google's servers (that needs a Google Cloud API setup, which is a lot of overhead for a small shop). Instead, in Google Sheets go to File → Download → Microsoft Excel (.xlsx) and upload that file below. For exports, download the .xlsx from any Report page, then in Google Sheets use File → Import → Upload to bring it in.
